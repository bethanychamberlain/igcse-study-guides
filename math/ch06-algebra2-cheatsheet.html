Key Vocabulary

Word What It Means Example
Variable A letter that stands for a number you don't know yet. x, y, n
Coefficient The number in front of a variable. In 3x, the coefficient is 3
Term A single chunk of an expression, separated by + or − signs. In 3x + 5y − 2, the terms are 3x, 5y, and −2
Expression A collection of terms. No equals sign. 3x + 5y − 2
Equation An expression with an equals sign. Something = something. 3x + 2 = 14
Simultaneous "At the same time" — two equations that must both be true for the same values of x and y. 3x − 2y = 3 and x + 4y = 8
Factorise Write an expression as a product of factors (put it into brackets). 6x + 9 = 3(2x + 3)
nth term A formula that gives you any term in a sequence based on its position n. 4n + 3 gives: 7, 11, 15, 19...
Common difference The fixed amount added each time in an arithmetic sequence. In 7, 11, 15, 19... the common difference is 4
Gradient How steep a line is. Rise divided by run. The "m" in y = mx + c. y = 2x + 3 has gradient 2
y-intercept Where the line crosses the y-axis. The "c" in y = mx + c. y = 2x + 3 crosses at y = 3

Method Cards — Step-by-Step

1. Factorising
1 Find the highest common factor (HCF) of all terms (numbers AND letters).
2 Write the HCF outside the bracket.
3 Divide each term by the HCF. Write results inside the bracket.
4 CHECK: expand — you should get the original expression back.
12x2 + 8x → HCF = 4x → 4x(3x + 2)
2. Simultaneous Equations (Elimination)
1 Label equations (1) and (2).
2 Multiply to match coefficients of one variable.
3 Same signs? Subtract. Different signs? Add.
4 Solve for the remaining variable.
5 Substitute back into either original equation to find the other variable.
6 CHECK: put both values into the OTHER equation.
3x − 2y = 3, x + 4y = 8 → ×3: 3x + 12y = 24; subtract: 14y = 21; y = 1.5, x = 2
3. Finding the nth Term
1 Find the common difference (d) = second term − first term.
2 Formula starts with dn.
3 Find the adjustment: first term − d = what you add or subtract.
4 nth term = dn + adjustment
5 CHECK: plug in n = 1, 2, 3 to verify.
7, 11, 15, 19... → d = 4, adjustment = 7 − 4 = 3 → nth term = 4n + 3
4. Rearranging Formulae
1 Identify which letter to isolate.
2 Undo operations in reverse order: +/− first, then ×/÷, then powers.
3 Do the same operation to both sides each time.
4 One new line per step — never skip.
y = 3x + 7, make x the subject: → y − 7 = 3x → x = (y − 7)/3
5. Algebraic Fractions (Add/Subtract)
1 Find the lowest common denominator (LCD).
2 Multiply top and bottom of each fraction to get the LCD.
3 Add or subtract the tops only. Keep the bottom the same.
4 Use brackets when subtracting — the minus applies to the WHOLE top.
5 Simplify if possible.
3x/4 − (x−1)/2 → LCD = 4 → 3x/4 − 2(x−1)/4 → (3x − 2x + 2)/4 → (x+2)/4
6. Linear Graphs (y = mx + c)
1 m = gradient (slope). c = y-intercept.
2 To plot: make a table with at least 3 values of x.
3 Calculate y for each x using the equation.
4 Plot points, join with a ruler.
5 Gradient from two points: (y2 − y1) / (x2 − x1)
y = 2x + 1: gradient = 2 (up 2 for every 1 right), crosses y-axis at 1

Common Mistakes — Don't Get Caught

If You Wrote This... You Probably Meant This... Topic
3xy − 2x = 3x(y − 2) x(3y − 2) — the HCF is x, not 3x Factorising
Subtracting equations but getting the sign wrong Remember: −(−2y) = +2y. Subtracting a negative flips the sign. Simultaneous
nth term = 4n for sequence 7, 11, 15... 4n + 3 — always check: when n=1, 4(1) = 4, not 7. You need the +3. Sequences
y = 3x + 7 → x = y/3 + 7 x = (y − 7)/3 — undo the +7 FIRST, then divide. The whole expression divides by 3. Rearranging
3x/4 − (x−1)/2 → (3x − x − 1)/4 (3x − 2x + 2)/4 — the minus distributes to BOTH terms: −(x−1) = −x+1. Then multiply by 2. Fractions
Gradient of y = 5 − 2x is 5 Gradient is −2. Rewrite as y = −2x + 5. The number with x is the gradient. Graphs
Forgetting to check the answer Always substitute back. Every topic has a check method. Use it. Everything

Memory Tricks

Simultaneous Equations — Same Sign Subtract
Same Sign? Subtract.   Different signs? aDd.   (Three S's vs. two D's.)
nth Term Formula
The common difference is always the number in front of n.   The adjustment is first term minus d.   "dn + a" = Difference times n, plus Adjustment.
Rearranging — Peel the Onion
Think of the operations around your variable as layers of an onion. The outside layer comes off first. Addition/subtraction is the outer layer. Multiplication/division is next. Powers/roots are the core.
y = mx + c
"m" = "mountain" (how steep).   "c" = "crosses" the y-axis.
Algebraic Fractions — Bracket the Minus
Whenever you subtract a fraction, put the entire numerator in brackets first, then distribute the minus. This one habit prevents 90% of fraction errors.

Quick Worked Examples

Factorise 6a − 9ab = 3a(2 − 3b)   [HCF = 3a]
Simultaneous 2x + y = 7, x − y = 2 → Add (different signs): 3x = 9, x = 3, y = 1. Check: 3 − 1 = 2.
nth term 5, 9, 13, 17... → d = 4, adjustment = 5 − 4 = 1 → nth term = 4n + 1
Rearrange A = ½bh, make h subject → 2A = bh → h = 2A/b
Alg. fraction x/3 + x/5 = 5x/15 + 3x/15 = 8x/15
Gradient Through (2, 5) and (6, 13): (13−5)/(6−2) = 8/4 = 2

Quick-Fire Exam Cues

If the Question Says... You Need To...
"Factorise completely" Take out the highest common factor (numbers AND letters). "Completely" means don't leave anything that can still be factored.
"Solve these simultaneous equations" Use elimination or substitution. Find values of BOTH variables. Always check in the other equation.
"Find the nth term" Find d (common difference), then write dn + adjustment. Verify with n = 1, 2, 3.
"Make x the subject" or "Rearrange for x" Get x completely on its own on one side. Undo operations in reverse order.
"Simplify" (with fractions) Get a common denominator, combine the tops, then cancel if possible.
"Find the gradient" Either read m from y = mx + c, or calculate rise/run from two points.
"Draw the graph of y = ..." Make a table of values (at least 3 points), plot them, join with a ruler.
"Show your working" They mean it. One step per line. Even if you get the answer wrong, working = marks.
Exam Trap — Read the Question Twice
"Factorise" and "Expand" are opposites. Factorise = put INTO brackets. Expand = take OUT of brackets. Read the question carefully — doing the wrong one scores zero.
