Key Vocabulary

Word What It Means Don't Confuse With
Sequence A list of numbers that follow a rule. —
nth term A formula that gives you any term based on its position number n. The "term" itself (the value) vs. "n" (the position)
Common difference The fixed amount added each time in a linear sequence. The first term (it's not the same thing)
Equation Has an equals sign. Something = something. You solve it to find the unknown. Expression (no equals sign, can't be "solved")
Inverse operation The opposite: + undoes −, × undoes ÷. —
Gradient (m) How steep a line is. Rise ÷ run. The number in front of x in y = mx + c. y-intercept (that's the c, not the m)
y-intercept (c) Where the line crosses the y-axis. The value of y when x = 0. Gradient (the c is where it crosses, not how steep it is)
Substitute Replace a letter with a number and calculate the result. —

Method Cards — Step-by-Step

1. Finding the nth Term (Linear Sequence)
1 Find the common difference (d): second term − first term.
2 Formula starts with dn.
3 Find the adjustment: first term − d.
4 Write: nth term = dn + adjustment.
5 CHECK: put n = 1, 2, 3 into the formula — you must get the original sequence.
5, 9, 13, 17... → d = 4, adjustment = 5 − 4 = 1 → nth term = 4n + 1
2. Solving a Two-Step Equation
1 Look at the equation: something is happening to x.
2 Undo in reverse order: deal with +/− first, then ×/÷.
3 Whatever you do to one side, do the same to the other side.
4 CHECK: substitute your answer back into the original equation.
4x − 3 = 17 → add 3: 4x = 20 → divide by 4: x = 5. Check: 4(5) − 3 = 17.
3. Solving Equations with x on Both Sides
1 Collect x terms on one side: subtract the smaller x term from both sides.
2 Then solve the two-step equation as normal.
3 CHECK: substitute into both sides of the original equation. They must be equal.
5x + 2 = 3x + 10 → subtract 3x: 2x + 2 = 10 → subtract 2: 2x = 8 → x = 4
4. Equations with Brackets
1 Expand the brackets first: multiply the number outside by each term inside.
2 Then solve as a normal equation.
3 Be extra careful with negative signs: −2(x − 3) = −2x + 6, NOT −2x − 6.
3(2x − 1) = 15 → expand: 6x − 3 = 15 → add 3: 6x = 18 → x = 3
5. Drawing a Graph from an Equation
1 Make a table of values: list x values (usually −3 to 3 or as given).
2 Substitute each x into the equation to find y.
3 Plot the (x, y) points on the grid with a small cross.
4 Join with a ruler (for straight lines) or a smooth curve.
5 CHECK: if all points lie on the line, your calculations are right. If one doesn't fit, recheck that row.
y = 2x + 1: x = −1 → y = −1, x = 0 → y = 1, x = 1 → y = 3, x = 2 → y = 5. Straight line.
6. Finding the Gradient from Two Points
1 Pick two points: (x1, y1) and (x2, y2).
2 Gradient = (y2 − y1) ÷ (x2 − x1) = rise ÷ run.
3 Keep the order consistent — if you subtract "second minus first" for y, do the same for x.
(2, 5) and (6, 13): rise = 13 − 5 = 8, run = 6 − 2 = 4, gradient = 8/4 = 2

Inverse Operations — Know Your Opposites

Operation Inverse (Opposite) Example
+ (add) − (subtract) x + 7 = 12 → subtract 7
− (subtract) + (add) x − 3 = 10 → add 3
× (multiply) ÷ (divide) 3x = 18 → divide by 3
÷ (divide) × (multiply) x/4 = 5 → multiply by 4

Say It in One Sentence

Sequence A list of numbers with a pattern — your job is to spot the rule and write the formula.
nth term It's always dn + adjustment for linear sequences — two numbers to find, that's it.
Solving equations Whatever you do to one side, do to the other — peel away operations in reverse order until x is alone.
Gradient Rise over run — how much y goes up (or down) for every 1 that x goes across.
y = mx + c m tells you the steepness (positive = uphill, negative = downhill), c tells you where it crosses the y-axis.

Memory Tricks

nth Term — "d n + a"
Difference times n, plus Adjustment. Find d (what you add each time). Then the adjustment = first term minus d. Done. Two numbers, every time.
Equations — "Balance the Scales"
Picture a set of scales with x on one side. To keep them balanced, every operation you do must happen on both sides. If you add 3 to the left, add 3 to the right. Simple as that.
Negative Signs in Brackets — "Flip Everything Inside"
When there's a minus sign before a bracket, every sign inside flips: −(x − 3) becomes −x + 3. Minus times minus = plus. This is the single most common algebra mistake — slow down here.
y = mx + c — "m = Mountain, c = Crosses"
m = how steep the mountain is (gradient). c = where it crosses the y-axis. If the equation is in a weird order (like y = 5 − 2x), rewrite it as y = −2x + 5 to read off m and c.
Always Check Your Answer
Sequences: plug n = 1, 2, 3 into your formula. Equations: substitute your answer back in. Graphs: do all points sit on the line? Gradient: does the sign match the direction (uphill = positive)? Checking takes 15 seconds and catches mistakes before the examiner does.

Quick-Fire Exam Cues

If the Question Says... The Answer Is...
"Find the nth term" Find d (common difference), then write dn + adjustment. Verify with n = 1, 2, 3.
"Find the 50th term" Write the nth term formula first, then substitute n = 50.
"Solve" Find the value of x. Use inverse operations, one step per line.
"Draw the graph of y = ..." Make a table of at least 3 values, plot, join with ruler.
"Find the gradient" Either read m from y = mx + c, or use rise ÷ run from two points.
"Find the equation of the line" Find gradient (m), find where it crosses y-axis (c), write y = mx + c.
"Show your working" They mean it. One step per line. Wrong answer + correct working = partial marks.
Negative x-values in a table Be very careful with signs. 2 × (−3) = −6. If a point doesn't fit the line, recheck that calculation.
Exam Trap — Gradient Order
When calculating gradient from two points, subtract in the same direction for both x and y. (13−5)/(6−2) = 8/4 = 2. But (13−5)/(2−6) = 8/(−4) = −2 (WRONG!). Stay consistent.
