Key Vocabulary

Word What It Means Don't Confuse With
Investigation An extended problem where you explore, find patterns, and write rules. No single "right method." A standard calculation (those have one fixed method)
Pattern A regular relationship between numbers in a sequence or table. —
Generalise Write a formula or rule that works for ANY case, not just the ones you tested. Just listing examples (that's not generalising)
Systematic Working through cases in order (1, 2, 3, 4...) rather than jumping around randomly. Random trial and error (that misses patterns)
Linear pattern A pattern where the differences between terms are constant. Rule looks like dn + c. Quadratic pattern (differences are NOT constant — second differences are)
Constant difference When you subtract consecutive terms and always get the same number. —
Prediction Using your rule to calculate a result you haven't tested yet. Guess (a prediction is based on your formula, not a guess)
Verify / Test Checking that your rule works by trying it on additional cases. Prove (testing =/= proving; proving requires algebra for ALL cases)

The 5-Step Problem-Solving Process

  1. Understand the problem. Read it twice. What are you asked? What information do you have? What are you looking for?
  2. Try a simpler case. If the problem uses big numbers or complicated shapes, try it with small, easy numbers first (2×2 grid before 10×10).
  3. Be systematic. Organise your results in a table. Don't jump around randomly — go in order (n = 1, 2, 3, 4...).
  4. Look for a pattern. Check the differences between consecutive results. Constant difference = linear rule (dn + c).
  5. Generalise. Write a formula that works for any n. Test it on cases you already tried. Then predict a new case and check it.

How to Spot What Kind of Pattern It Is

What You See Type of Pattern Rule Looks Like
Constant 1st differences Linear dn + c   (e.g. 3n + 2)
Constant 2nd differences Quadratic an2 + bn + c
Each term is multiplied by a constant Geometric / exponential a × rn
Results involve squares or products Look for n2 or n(n+1) Check by squaring n
The Difference Method
Write out your results. Subtract each from the next to get 1st differences. If those aren't constant, subtract them again to get 2nd differences. Constant 1st differences = linear. Constant 2nd differences = quadratic.

Comparison Problems — The Method

Step What to Do
1. Identify variables What changes? (e.g. km driven, days hired, number of items)
2. Calculate for each option Work out the total cost/value for each choice
3. Try multiple scenarios Don't just check one case — try low, medium, and high values
4. State your conclusion Say which is best AND for what situation, with numbers to support it
Exam Trap — "Explain" Means Show Numbers
"Company A is cheaper" gets 0 marks. "Company A costs $102.50 while Company B costs $170, so A is cheaper by $67.50" gets full marks. Always show calculations in your explanation.

Memory Tricks

"If It's Too Hard, Make It Easier"
10×10 grid? Try 2×2 first. 100 coins? Try 3 coins first. The single most powerful strategy in all of maths problem-solving: reduce to a simpler case and let the pattern show itself.
"Table = Treasure Map"
A results table is like a treasure map — the pattern (treasure) is hidden in the numbers. Without a table, you're searching blindly. With a table, the pattern practically jumps out at you.
Constant Difference? It's Linear.
If every pair of consecutive results differs by the same amount (say, 3), your rule is 3n + something. To find the "something," substitute n = 1 and solve.
"Show Working = Show Money"
Think of each line of working as earning money. Even if your final answer is wrong, correct working steps still earn marks. Never skip steps — each step is a mark the examiner can give you.
The Investigation Checklist — STPREV
Simpler cases. Table of results. Pattern spotted. Rule written. Extra cases tested. Verify and explain.

Quick-Fire Exam Cues

If the Question Says... The Answer Is...
"Investigate..." Try simple cases, make a table, find a pattern, write a rule, test it, explain why it works.
"Find a rule connecting n and..." Look at differences. Constant difference d → rule is dn + c. Find c by substituting n = 1.
"Test your rule for n = 6" Substitute n = 6 into your formula AND work out the real answer. Show both match.
"Predict the result for n = 20" Substitute n = 20 into your formula. State the answer clearly.
"Explain why..." Use numbers and calculations, not just words. Show the working that proves your point.
"Which option is cheaper / better?" Calculate both options with numbers. Compare them. State which is better AND by how much.
"Estimate the answer to 48.7 × 3.2" Round to 1 s.f.: 50 × 3 = 150. Use ≈ symbol.
"Extend the investigation / What happens if...?" Change one variable (e.g. try a different grid size, different shape). Repeat the whole process: table, pattern, rule.
