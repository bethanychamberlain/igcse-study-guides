Section 6.1 Factorising Expressions
Factorising is the opposite of expanding brackets. You take an expression and rewrite it with brackets by pulling out the common factor. Think of it as "un-multiplying."
W
Write This Down — What Factorising Means

Factorising means finding what's shared by every term, taking it outside a bracket, and writing what's left inside the bracket.

If you expand the bracket and get back to the original expression, you did it right.

W
Write This Down — Step-by-Step Method

Step 1: Look at each term. What number and/or letter appears in ALL of them?

Step 2: Write that common factor outside the bracket.

Step 3: Divide each original term by the common factor. Write the results inside the bracket.

Step 4: CHECK — expand the bracket. You should get back to the original expression.

Example: Factorise 3xy − 2x

1.Terms are 3xy and 2x. Both contain x.
2.Common factor = x. Write it outside: x( ... )
3.3xy ÷ x = 3y.   2x ÷ x = 2.   Keep the minus sign!
4.Answer: x(3y − 2)
Check:x × 3y = 3xy.   x × (−2) = −2x.   3xy − 2x. Correct!
T
Study Tip — Always Check by Expanding

Every single time you factorise, expand your answer to check. This is your safety net. Even on the exam, if you have 30 seconds spare, expand to verify. If it doesn't match the original, you made an error.

!
Don't Confuse — Signs Matter

Watch the minus signs carefully. If the original expression has a minus, the minus goes inside the bracket.

6a − 9ab = 3a(2 − 3b) ← the minus is inside, not outside.

Check: 3a × 2 = 6a. 3a × (−3b) = −9ab. Correct!

W
Write This Down — Finding the Biggest Common Factor

Always take out the highest common factor (HCF), not just any common factor.

Example: Factorise 12x2 + 8x

1.Numbers: HCF of 12 and 8 = 4. Letters: both have at least one x.
2.Common factor = 4x
3.12x2 ÷ 4x = 3x.   8x ÷ 4x = 2.
4.Answer: 4x(3x + 2)
Check:4x × 3x = 12x2.   4x × 2 = 8x.   12x2 + 8x. Correct!
  • I can explain what factorising means in my own words
  • I can find the common factor in an expression
  • I can factorise a two-term expression and check by expanding
  • I know to watch the minus signs carefully
Section 6.2 Simultaneous Equations
"Simultaneous" means "at the same time." You have two equations and two unknowns. You need to find the values of both unknowns that make both equations true at the same time.
?
Why This Matters

Simultaneous equations are worth 4–5 marks on the IGCSE exam. They appear almost every year. Learn the method and you pick up easy marks — the steps are always the same.

W
Write This Down — The Elimination Method (Step by Step)

The goal: make one variable disappear so you only have one unknown left.

Step 1: Label the equations. Call them (1) and (2).

Step 2: Make the coefficients match. Multiply one or both equations so that the number in front of one variable is the same in both equations.

Step 3: Add or subtract. If the matching coefficients have the same sign, subtract one equation from the other. If they have different signs, add the equations.

Step 4: Solve. You now have one equation with one unknown. Solve it.

Step 5: Substitute back. Put the value you found into either original equation to find the other variable.

Step 6: CHECK. Put both values into the other original equation. If it works, you're right.

W
Write This Down — Full Worked Example

Solve simultaneously:   3x − 2y = 3   ... (1)   and   x + 4y = 8   ... (2)

1.Label: Equation (1): 3x − 2y = 3.   Equation (2): x + 4y = 8.
2.Match coefficients of x: Multiply equation (2) by 3 so both have 3x.
(2) × 3:   3x + 12y = 24   ... call this (3)
3.Subtract: Both have +3x (same sign), so subtract.
(3) − (1):   (3x + 12y) − (3x − 2y) = 24 − 3
3x + 12y − 3x + 2y = 21
14y = 21
4.Solve for y:   y = 21 ÷ 14 = 1.5
5.Substitute y = 1.5 into (2):   x + 4(1.5) = 8  →  x + 6 = 8  →  x = 2
6.Check in (1):   3(2) − 2(1.5) = 6 − 3 = 3. Correct!
Answer: x = 2, y = 1.5
!
Don't Confuse — When to Add vs. Subtract

"Same sign? Subtract.   Different signs? Add."

If both equations have +2y, you subtract one from the other to eliminate y.

If one has +2y and the other has −2y, you add them together to eliminate y.

Memory trick: Same Sign Subtract. Different signs aDd.

T
Study Tip — Write Every Single Step

Do NOT try to do simultaneous equations in your head. Write out every line. On the exam, even if your final answer is wrong, you get marks for correct working. One line per step = maximum marks.

W
Write This Down — The Substitution Method (Alternative)

Use this when one equation already has a variable on its own (like x = ... or y = ...).

Step 1: Rearrange one equation to get a variable by itself: e.g., x = 8 − 4y

Step 2: Replace that variable in the other equation with the expression.

Step 3: Solve the resulting equation (it'll only have one variable).

Step 4: Substitute back to find the other variable.

Same example using substitution:

1.From (2): x = 8 − 4y
2.Put into (1): 3(8 − 4y) − 2y = 3
3.24 − 12y − 2y = 3  →  24 − 14y = 3  →  −14y = −21  →  y = 1.5
4.x = 8 − 4(1.5) = 8 − 6 = 2
Same answer: x = 2, y = 1.5
  • I can solve simultaneous equations by elimination
  • I know when to add vs. subtract (same sign subtract, different signs add)
  • I can solve by substitution when one variable is already isolated
  • I always check my answer in the other equation
Section 6.3 Sequences and nth Term
A sequence is a list of numbers that follow a pattern. The nth term is a formula that lets you find any term in the sequence without listing them all out.
W
Write This Down — Key Definitions

Common difference (d) = the amount you add each time to get from one term to the next.

Arithmetic sequence = a sequence where the common difference is the same between every pair of terms.

nth term = a formula that gives you the value of any term. Written as something like 4n + 3 where n is the position number (1st, 2nd, 3rd...).

W
Write This Down — How to Find the nth Term

Step 1: Find the common difference (d). Subtract the first term from the second.

Step 2: The nth term starts with dn. (If d = 4, write 4n.)

Step 3: Work out what you need to add or subtract. Compare the first term to d × 1.

Step 4: Write the full formula: nth term = dn + (adjustment)

Step 5: CHECK — put n = 1, n = 2, n = 3 into your formula. You should get the original sequence.

Example: Find the nth term of: 7, 11, 15, 19, ...

1.Common difference: 11 − 7 = 4
2.So the formula starts with 4n
3.When n = 1: 4 × 1 = 4. But the first term is 7. So we need to add 3.
4.nth term = 4n + 3
Check:n=1: 4(1)+3 = 7.   n=2: 4(2)+3 = 11.   n=3: 4(3)+3 = 15.   n=4: 4(4)+3 = 19. All correct!
nth term = 4n + 3
!
Don't Confuse — n Is the Position, Not the Term

n is the position number: 1st, 2nd, 3rd, etc. It is NOT the value of the term.

In the sequence 7, 11, 15, 19 ... the 3rd term is 15. So when n = 3, the term = 15. Don't mix these up.

If they ask "What is the 50th term?", you put n = 50 into your formula: 4(50) + 3 = 203.

T
Study Tip — The Quick Way to Remember

The common difference is always the number in front of n. The adjustment is always what you add or subtract to make the first term work. Just two numbers to find.

  • I can find the common difference in a sequence
  • I can write the nth term formula
  • I can use the nth term to find the 50th or 100th term
  • I always check my formula against the first 3 terms
Section 6.4 Rearranging Formulae
Rearranging a formula means getting a different letter on its own on one side of the equals sign. The golden rule: whatever you do to one side, you must do to the other side.
W
Write This Down — The Golden Rule

An equation is like a set of scales. Both sides must stay balanced. If you add 5 to one side, you add 5 to the other. If you divide one side by 3, you divide the other by 3.

The operations you use to rearrange are the opposite (inverse) of what's currently happening:

If the letter is being...You undo it by...
Added (+)Subtracting (−) from both sides
Subtracted (−)Adding (+) to both sides
Multiplied (×)Dividing (÷) both sides
Divided (÷)Multiplying (×) both sides
Squared (2)Square rooting both sides
W
Write This Down — Step-by-Step Method

Step 1: Identify which letter you want to make the subject (get on its own).

Step 2: Look at what's happening to that letter. Undo each operation in reverse order (undo addition/subtraction first, then multiplication/division, then powers).

Step 3: Do the same operation to both sides each time.

Step 4: Keep going until the letter is on its own.

Example: Make x the subject of: y = 3x + 7

1.Want: x on its own. Currently x is multiplied by 3, then 7 is added.
2.Undo the +7 first:   y − 7 = 3x
3.Undo the ×3:   (y − 7) ÷ 3 = x
x = (y − 7) / 3
!
Don't Confuse — Order Matters When Rearranging

Undo operations in reverse order of BIDMAS/BODMAS:

1. Undo addition/subtraction first

2. Then undo multiplication/division

3. Then undo powers/roots last

This is the opposite order from how you'd calculate. Think of it like peeling off layers — the last thing that was done to x is the first thing you undo.

W
Write This Down — Harder Example with Squares

Example: I = M(k2 + c2).   Rearrange to make k the subject.

1.Want k on its own. Currently: k is squared, c2 is added, then multiplied by M.
2.Undo ×M:   I / M = k2 + c2
3.Undo +c2:   (I / M) − c2 = k2
4.Undo the square (take square root):   k = √( I/M − c2 )
k = √( I/M − c2 )
T
Study Tip — Write Every Step on a New Line

Never try to rearrange in one jump. Write a new line for each operation you do. This way, if you make a mistake, you can spot exactly where it went wrong. Also, the examiner can give you method marks for each correct step.

  • I understand the "scales" rule — same to both sides
  • I can rearrange a simple formula like y = mx + c to make x the subject
  • I know to undo operations in reverse order
  • I can handle formulae with squares and square roots
Section 6.5 Algebraic Fractions
Algebraic fractions work exactly like normal fractions — same rules for simplifying, adding, and subtracting. The only difference is that the numbers are replaced by letters and expressions.
W
Write This Down — Simplifying Algebraic Fractions

To simplify: cancel any factor that appears in both the top and the bottom.

Example: Simplify (p/t) ÷ (2/t)

1.Dividing by a fraction = flip it and multiply:   (p/t) × (t/2)
2.Multiply tops: p × t = pt.   Multiply bottoms: t × 2 = 2t.
3.Cancel t from top and bottom:   pt / 2t = p/2
Answer: p/2
W
Write This Down — Adding and Subtracting Algebraic Fractions

Same rule as normal fractions: you need a common denominator.

Step 1: Find the lowest common denominator (LCD).

Step 2: Multiply top and bottom of each fraction to get that denominator.

Step 3: Add or subtract the tops. Keep the denominator the same.

Step 4: Simplify if possible.

Example: Simplify   3x/4 − (x−1)/2

1.LCD of 4 and 2 = 4
2.First fraction already has denominator 4:   3x/4
Second fraction: multiply top and bottom by 2:   2(x−1)/4
3.Subtract the tops:   [3x − 2(x−1)] / 4
4.Expand the bracket:   [3x − 2x + 2] / 4 = (x + 2) / 4
Answer: (x + 2) / 4
!
Don't Confuse — Brackets When Subtracting

When you subtract a fraction, the minus sign applies to the entire top, not just the first term. Use brackets!

3x/4 − (x−1)/2  →  the minus hits both the x AND the −1.

−(x − 1) = −x + 1   (NOT −x − 1)

This is the number one mistake on algebraic fractions. Always use brackets when subtracting.

  • I can simplify algebraic fractions by cancelling common factors
  • I remember that dividing = flip and multiply
  • I can add/subtract algebraic fractions using a common denominator
  • I use brackets when subtracting fractions to avoid sign errors
Section 6.6 Linear Graphs
A linear graph is a straight line. Every straight line can be described by the equation y = mx + c, where m is the gradient (slope) and c is the y-intercept (where the line crosses the y-axis).
W
Write This Down — y = mx + c

m = the gradient (how steep the line is). A positive m means the line goes uphill left to right. A negative m means downhill.

c = the y-intercept (where the line crosses the y-axis). This is the value of y when x = 0.

Example: y = 2x + 3

Gradient (m) = 2 → for every 1 you go right, the line goes up by 2.
y-intercept (c) = 3 → the line crosses the y-axis at (0, 3).
W
Write This Down — Finding the Gradient

Gradient = rise / run = (change in y) / (change in x)

Pick two points on the line, say (x1, y1) and (x2, y2).

Gradient = (y2 − y1) / (x2 − x1)

Example: Find the gradient of the line through (1, 3) and (4, 9).

1.Rise: 9 − 3 = 6
2.Run: 4 − 1 = 3
3.Gradient = 6 / 3 = 2
W
Write This Down — How to Plot a Linear Graph

Step 1: Make a table of values. Pick 3 or 4 values for x (e.g., 0, 1, 2, 3).

Step 2: Substitute each x into the equation to find y.

Step 3: Plot each (x, y) point on the grid.

Step 4: Join the points with a straight line using a ruler.

Example: Plot y = 2x + 1

x2x + 1y
02(0) + 11
12(1) + 13
22(2) + 15
32(3) + 17

Plot points: (0,1), (1,3), (2,5), (3,7). Join with a ruler. Done.

!
Don't Confuse — Gradient vs. y-Intercept

In y = 3x + 5: the gradient is 3 (the number attached to x), the y-intercept is 5 (the number on its own).

In y = −2x + 7: the gradient is −2 (negative = line goes downhill), the y-intercept is 7.

In y = x − 4: the gradient is 1 (there's a hidden 1 in front of x), the y-intercept is −4.

T
Study Tip — Always Use a Ruler

On the exam, a wobbly freehand line loses marks. Always use a ruler. And plot at least 3 points — if one is wrong, you'll notice because it won't line up with the others.

  • I can identify gradient and y-intercept from y = mx + c
  • I can calculate gradient from two points
  • I can make a table of values and plot a straight line
  • I know to use a ruler and plot at least 3 points
Putting It All Together Key Study Strategies for Algebra
T
The Number One Rule

Write every step down. Algebra is not a subject you can do in your head. The students who lose marks are the ones who try to skip steps. The students who get full marks write out every single line, even when it feels "obvious." This also means the examiner can give you partial marks if you make a small error at the end.

T
Letters Are Just Numbers You Don't Know Yet

If algebra feels confusing, try replacing the letters with actual numbers first to understand what's happening. For example, if you need to rearrange y = 3x + 7, try it with y = 3(2) + 7 = 13 first. See how the operations work with real numbers, then do it with letters.

T
Check, Check, Check

Every topic in this chapter has a way to check your answer:

• Factorising: Expand the brackets — should get the original.

• Simultaneous equations: Put both values into the OTHER equation.

• nth term: Plug n=1, n=2, n=3 into your formula.

• Rearranging: Substitute a number in and see if both versions give the same result.

• Fractions: Try with simple numbers to see if your answer makes sense.
