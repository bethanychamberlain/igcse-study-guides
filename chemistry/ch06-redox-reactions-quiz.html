Foundation — Can You Name It?

Worth 1-2 marks each on the exam. Get these right and you're building a solid base.
1
Define oxidation and reduction in terms of oxygen.
2 marks
Answer
Oxidation is the gain of oxygen [1]. Reduction is the loss of oxygen [1].
One mark for each correct definition. The exam wants the words "gain" and "loss" paired with "oxygen" — don't mix them up.

2
What does the mnemonic OIL RIG stand for? Define oxidation and reduction in terms of electrons.
2 marks
Answer
OIL: Oxidation Is Loss (of electrons) [1]. RIG: Reduction Is Gain (of electrons) [1].
The electron definition is the one the exam uses most often. If you only learn one thing from this chapter, learn OIL RIG.

3
What is the oxidation number of any uncombined element? Give an example.
2 marks
Answer
The oxidation number of any uncombined element is 0 (zero) [1]. Example: Fe (iron metal) has an oxidation number of 0 [1]. (Any valid example accepted: Na, O₂, Cl₂, Mg, etc.)
This is Rule 3 from the textbook. "Uncombined" means the element is on its own, not in a compound.

4
Define oxidising agent and reducing agent.
2 marks
Answer
An oxidising agent is a substance that oxidises another substance / takes electrons from another substance (and is itself reduced) [1]. A reducing agent is a substance that reduces another substance / gives electrons to another substance (and is itself oxidised) [1].
Remember: the oxidising agent is itself reduced, and the reducing agent is itself oxidised. The agent does the opposite of what happens to it.

Intermediate — Can You Explain It?

Worth 3-4 marks each. You need to explain WHY using definitions and examples, not just name things.
5
In the reaction below, identify which substance is oxidised and which is reduced. Explain your answer.

CuO (s) + H₂ (g) → Cu (s) + H₂O (l)
4 marks
Answer
CuO is reduced [1] because it loses oxygen / copper's oxidation number decreases from +II to 0 [1].
H₂ is oxidised [1] because it gains oxygen / hydrogen's oxidation number increases from 0 to +I [1].
You get one mark for correctly identifying each substance, and one mark for explaining each. You can use either the oxygen definition or the oxidation number definition — both are accepted. Just be consistent.
Rate your answer:
6
Write the balanced half-equations for the reaction between magnesium and oxygen. Label which half-equation shows oxidation and which shows reduction.

2Mg (s) + O₂ (g) → 2MgO (s)
4 marks
Answer
Oxidation: 2Mg → 2Mg²⁺ + 4e⁻ [1] (magnesium loses electrons) [1]
Reduction: O₂ + 4e⁻ → 2O²⁻ [1] (oxygen gains electrons) [1]
Key checks: (1) Oxygen must be written as O₂, not O. (2) The number of electrons must be the same in both equations (4e⁻ here). (3) Make sure charges balance on each side.

7
Give the oxidation number of each element in the following compounds:
(a) NaOH    (b) CaCl₂    (c) Fe₂O₃
3 marks
Answer
(a) NaOH: Na = +I, O = −II, H = +I [1]
(b) CaCl₂: Ca = +II, Cl = −I [1]
(c) Fe₂O₃: Fe = +III, O = −II [1]
Use the "total adds up to zero" rule to check: (a) (+I) + (−II) + (+I) = 0. (b) (+II) + 2(−I) = 0. (c) 2(+III) + 3(−II) = +6 − 6 = 0. If your numbers don't add to zero, something is wrong.

Advanced — Can You Discuss It?

Worth 5-6 marks each. These need full sentences, multiple points, and clear reasoning. This is where exam grades are won or lost.
8
Chlorine gas is bubbled through a colourless solution of potassium bromide. The solution turns orange.

Cl₂ (g) + 2KBr (aq) → 2KCl (aq) + Br₂ (aq)

(a) Write the balanced half-equations for this reaction and label each as oxidation or reduction. [3]
(b) Combine the half-equations to write the ionic equation. [1]
(c) Identify the oxidising agent and the reducing agent. [2]
6 marks
Answer
(a)
Reduction: Cl₂ + 2e⁻ → 2Cl⁻ [1] (chlorine gains electrons) [1]
Oxidation: 2Br⁻ → Br₂ + 2e⁻ [1] (bromide loses electrons)

(b) Ionic equation: Cl₂ + 2Br⁻ → 2Cl⁻ + Br₂ [1] (the electrons cancel when you add the half-equations)

(c) The oxidising agent is Cl₂ (chlorine) because it takes electrons from the bromide ions / is itself reduced [1]. The reducing agent is KBr / Br⁻ (bromide ion) because it gives electrons to chlorine / is itself oxidised [1].
This is a displacement reaction — chlorine displaces bromine because chlorine is more reactive (stronger oxidising agent). The colour change from colourless to orange is because Br₂ (bromine) is produced, which is orange in solution.

9
The equation for the combustion of methane is:

CH₄ (g) + 2O₂ (g) → CO₂ (g) + 2H₂O (l)

(a) Use oxidation numbers to show that this is a redox reaction. [3]
(b) Identify the substance that is oxidised and the substance that is reduced. [2]
5 marks
Answer
(a)
In CH₄: C = −IV, H = +I [1]
In CO₂: C = +IV, O = −II
In O₂: O = 0 [1]
Carbon's oxidation number changes from −IV to +IV (increases). Oxygen's oxidation number changes from 0 to −II (decreases). Since oxidation numbers change, this is a redox reaction [1].

(b) Carbon (in CH₄) is oxidised because its oxidation number increases from −IV to +IV [1]. Oxygen is reduced because its oxidation number decreases from 0 to −II [1].
This question tests whether you can apply oxidation numbers to a compound like CH₄ where carbon is negative. Use the rule: H = +I in compounds, and the total must equal zero. So in CH₄: C + 4(+I) = 0, therefore C = −IV.

10
A student has two unknown solutions, X and Y. She suspects one contains a reducing agent and the other contains an oxidising agent.

(a) Describe how she could test solution X for the presence of a reducing agent. State what she would observe if a reducing agent is present. [3]
(b) Describe how she could test solution Y for the presence of an oxidising agent. State what she would observe if an oxidising agent is present. [3]
6 marks
Answer
(a) Test for a reducing agent:
Add acidified potassium manganate(VII) solution to solution X [1]. Potassium manganate(VII) is purple [1]. If a reducing agent is present, the purple colour will fade / become colourless [1]. (The Mn is reduced from +VII to +II.)

(b) Test for an oxidising agent:
Add potassium iodide solution to solution Y [1]. Potassium iodide solution is colourless [1]. If an oxidising agent is present, the solution will turn red-brown [1]. (The iodide ions are oxidised to iodine: 2I⁻ → I₂.)
Three marks per test: name the reagent, state its original colour, state what colour change you would see. These two tests are named specifically in the syllabus — they are very likely to come up on the exam.
