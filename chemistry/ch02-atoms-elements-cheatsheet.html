Key Vocabulary

Word What It Means Don't Confuse With
Atom The smallest particle of matter that cannot be broken down further by chemical means. Molecule (two or more atoms bonded together)
Element A substance that contains only one kind of atom. E.g. sodium, carbon, sulfur. Compound (two or more different elements bonded together)
Compound Atoms of different elements chemically bonded together. You need a chemical reaction to separate them. Mixture (not bonded — you can separate by physical methods)
Mixture Different substances not bonded together. Separated by filtering, distillation, etc. Compound (bonded — needs a chemical reaction to separate)
Proton number Number of protons in the nucleus. Identifies the element. Also called atomic number. Nucleon number (protons + neutrons — always bigger)
Nucleon number Total number of protons + neutrons. Also called mass number. Proton number (just protons, not neutrons)
Isotopes Atoms of the same element, same protons, but different numbers of neutrons. Different elements (isotopes have the SAME proton number)
Electronic configuration How electrons are arranged in shells. Written as 2,8,1 etc. —
Relative atomic mass (Aᵣ) The average mass of an element's isotopes, compared to 1/12 of carbon-12. Nucleon number (that's for ONE atom; Aᵣ is an average and can be a decimal)

Exam Trap — Shorthand Notation
The big number on top is the nucleon number (mass number). The small number on the bottom is the proton number (atomic number). The big number is always bigger because it includes neutrons. Neutrons = nucleon number − proton number.

Electron Shell Capacity

Shell Maximum Electrons Example
1st (closest to nucleus) 2 Helium: 2
2nd 8 Neon: 2, 8
3rd 8 (when outer shell) Argon: 2, 8, 8
Key Link
Group number = outer shell electrons. Group I = 1 outer electron. Group VII = 7 outer electrons. Period number = number of occupied shells. Period 3 = 3 shells.

Say It in One Sentence

Atom The smallest particle of an element that still behaves like that element.
Element Only one type of atom — nothing else mixed in, no other elements bonded to it.
Compound Two or more different elements bonded together that you can only separate with a chemical reaction.
Proton number The identity card of an element — every element has a unique proton number, and changing it changes the element.
Neutral atom Equal protons (+) and electrons (−), so the charges cancel out to zero.
Isotopes Same element, same protons, different neutrons — same chemistry, different mass.
Aᵣ The weighted average mass of all isotopes — closer to the most abundant isotope.
Noble gases Full outer shell = stable = unreactive — they don't need to gain, lose, or share electrons.

Memory Tricks

Neutrons = Big Minus Small
Neutrons = nucleon number − proton number. The big number on top minus the small number on the bottom. For sodium: 23 − 11 = 12 neutrons. This formula appears on every exam paper.
Big Number On Top
In shorthand notation, the nucleon number (mass) goes on top, proton number on the bottom. Think: "mass is heavy, it sits on top." The mass number is always the bigger number.
2, 8, 8 Rule
Electron shells fill in order: 2, then 8, then 8. After 2,8,8 (argon), the next electrons start a new 4th shell: potassium is 2,8,8,1 and calcium is 2,8,8,2. Don't try to put more than 8 in the 3rd shell for the first 20 elements.
Isotopes React the Same
Isotopes have the same electrons, so they have the same chemistry. Carbon-12 and carbon-14 both burn the same way. Different mass, same reactions. The exam loves asking "why do isotopes have the same chemical properties?" — answer: same number of electrons / same electronic configuration.
Aᵣ Sanity Check
Your calculated Aᵣ must fall between the lightest and heaviest isotope masses, and be closer to the more abundant one. If it's outside that range, you made an error.

Quick-Fire Exam Cues

If the Question Says... The Answer Is...
"What is an element?" A substance that contains only one type of atom
"What is a compound?" Two or more different elements chemically bonded together
"What is the difference between a compound and a mixture?" A compound has atoms chemically bonded (needs a chemical reaction to separate). A mixture is not bonded (separated by physical methods like filtering)
"An atom has 17 protons and nucleon number 35. Find the number of neutrons and electrons." Neutrons = 35 − 17 = 18. Electrons = 17 (same as protons in a neutral atom). Electronic configuration: 2, 8, 7
"Why does an atom have no overall charge?" It has an equal number of protons (+) and electrons (−), so the charges cancel out
"What are isotopes?" Atoms of the same element with the same number of protons but different numbers of neutrons
"Why do isotopes have the same chemical properties?" They have the same number of electrons (same electronic configuration), and chemical reactions depend on outer shell electrons
"Calculate the relative atomic mass given isotope data" Aᵣ = (mass₁ × %₁ ÷ 100) + (mass₂ × %₂ ÷ 100). Check: answer must be between the two masses
"Why are noble gases unreactive?" They have a full outer shell of electrons, so they do not need to gain, lose, or share electrons
"How are elements in the same group similar?" They have the same number of outer shell electrons, so they have similar chemical properties
