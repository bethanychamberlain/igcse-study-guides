Redox Reactions

Pin this to your wall

Key Vocabulary

Word What It Means Don't Confuse With
Oxidation Gain of oxygen OR loss of electrons. Oxidation number increases. Reduction (the opposite!)
Reduction Loss of oxygen OR gain of electrons. Oxidation number decreases. Oxidation (the opposite!)
Redox reaction A reaction where both oxidation and reduction happen at the same time. —
Oxidising agent The substance that causes oxidation in something else. It takes electrons and is itself reduced. Reducing agent (causes reduction, is itself oxidised)
Reducing agent The substance that causes reduction in something else. It gives electrons and is itself oxidised. Oxidising agent (causes oxidation, is itself reduced)
Oxidation number A number tracking how many electrons an atom has lost/gained/shared in a compound. Written as Roman numerals. Charge on an ion (similar idea but oxidation number applies to atoms in compounds too)
Half-equation Shows the electron transfer for just one part of a redox reaction (either the oxidation or the reduction). Full equation (shows the complete reaction with electrons cancelling out)

Oxidation vs Reduction — Three Ways to Define

Oxidation Reduction
Oxygen definition Gain of oxygen Loss of oxygen
Electron definition Loss of electrons Gain of electrons
Oxidation number Number increases (goes up) Number decreases (goes down)
Mnemonic OIL — Oxidation Is Loss RIG — Reduction Is Gain
Exam Trap — OIL RIG Feels Backwards
Your brain says "gaining = good = oxidation." WRONG. Oxidation is loss of electrons. Write OIL RIG at the top of every redox question. Every. Single. Time.

Oxidising Agent vs Reducing Agent

Oxidising Agent Reducing Agent
What it does Causes oxidation in the other substance Causes reduction in the other substance
How? Takes electrons from the other substance Gives electrons to the other substance
What happens to it? Gets reduced itself Gets oxidised itself
Common examples O₂, Cl₂, KMnO₄, H₂O₂ H₂, Na, Mg, Zn, C, CO, KI
Lab test Add KI solution → turns red-brown Add acidified KMnO₄ → purple fades to colourless
Exam Trap — The Agent Does the Opposite
The oxidising agent gets reduced. The reducing agent gets oxidised. Think of a bully (oxidising agent): it takes electrons from the victim and gets "richer" (reduced). The victim loses electrons (is oxidised).

Say It in One Sentence

OIL RIG Oxidation Is Loss of electrons, Reduction Is Gain of electrons — the single most important mnemonic in chemistry.
Redox Oxidation and reduction always happen together — you can't have one without the other.
Redox without oxygen Na + Cl₂ → NaCl is redox (electrons transfer) even though there is zero oxygen in the equation.
Oxidation number = 0 Any uncombined element has an oxidation number of zero: Fe = 0, O₂ = 0, Na = 0.
Spotting redox Write oxidation numbers before and after. If any change, it's redox. If none change, it's not.
Ionic equations Add the two half-equations together — the electrons cancel out.
The flip trick The substance that IS reduced = the oxidising agent. The substance that IS oxidised = the reducing agent.

Memory Tricks

OIL RIG — Write It On Every Paper
Oxidation Is Loss. Reduction Is Gain. Write this at the top of every exam answer about redox. It's your lifeline.
Oxygen in "Oxidation"
The "O" in Oxidation matches the "O" in Oxygen. If something gains oxygen, it is oxidised. Simple.
Oxidation Numbers — Up = Oxidised
Think of a number line. Moving right (towards positive) = oxidation. Moving left (towards negative) = reduction. 0 → +II = oxidised. 0 → −II = reduced. Up = oxidised, down = reduced.
Key Oxidation Number Rules
Uncombined elements = 0. Simple ions = the charge. H = +I (usually). O = −II (usually). Group I = +I always. Group II = +II always. All numbers in a compound add up to zero.
Find the Agent: Flip It
Step 1: Which substance is oxidised? Which is reduced? (Use OIL RIG.) Step 2: Flip. The substance that is reduced = the oxidising agent. The substance that is oxidised = the reducing agent.
Lab Tests (Colour Changes)
Test for reducing agent: add acidified KMnO₄ (purple) → turns colourless. Test for oxidising agent: add KI (colourless) → turns red-brown. Purple to colourless = reducing agent present. Colourless to brown = oxidising agent present.

Quick-Fire Exam Cues

If the Question Says... The Answer Is...
"Define oxidation in terms of electrons" Loss of electrons (OIL — Oxidation Is Loss)
"Define reduction in terms of electrons" Gain of electrons (RIG — Reduction Is Gain)
"CuO + H₂ → Cu + H₂O. What is oxidised and what is reduced?" CuO is reduced (loses oxygen). H₂ is oxidised (gains oxygen). CuO is the oxidising agent. H₂ is the reducing agent
"Write the half-equations for Na + Cl₂" Oxidation: 2Na → 2Na⁺ + 2e⁻. Reduction: Cl₂ + 2e⁻ → 2Cl⁻. Electrons balance (2 each side)
"Is this reaction redox? NaOH + HCl → NaCl + H₂O" No. Check oxidation numbers: Na = +I both sides, H = +I both sides, O = −II both sides, Cl = −I both sides. Nothing changes. Not redox
"Is this reaction redox? Fe + S → FeS" Yes. Fe: 0 → +II (oxidised). S: 0 → −II (reduced). Oxidation numbers changed
"Identify the oxidising agent in: CuO + C → Cu + CO" CuO is the oxidising agent (it is itself reduced — Cu goes from +II to 0). Carbon is the reducing agent (it is itself oxidised — C goes from 0 to +II)
"Describe the test for a reducing agent" Add acidified potassium manganate(VII) solution. If a reducing agent is present, the purple colour fades to colourless
"Describe the test for an oxidising agent" Add potassium iodide solution. If an oxidising agent is present, the solution turns red-brown (iodine is formed)
"Give everyday examples of oxidation" Rusting (iron + oxygen + water → rust), combustion/burning (fuel gains oxygen), respiration (glucose + oxygen in your body)
